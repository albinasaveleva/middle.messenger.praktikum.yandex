# Веб-приложение «Чат»

## Технологии

 * JavaScript
 * HTML
 * CSS
 * PostCSS
 * Hаndlebars



## Разработка

### Сборка и запуск проекта 
Выполните команду:

`npm run start`

### Запуск проекта в режиме разработки
Выполните команду:

`npm run dev`

### Создание билда
Выполните команду:

`npm run build`
